# Ex.No:3(b) POLYMORPHISM

## QUESTION:
Write a Java program to create a base class Animal (Animal Family) with a method called Sound(). Create two subclasses Bird and Cat. Override the Sound() method in each subclass to make a specific sound for each animal.

## AIM:

To write a Java program to demonstrate runtime polymorphism using method overriding.
## ALGORITHM :
1. Start the program.
2. Create a base class Animal and derived classes Bird and Cat with overridden sound() methods.
3. Read the animal name from the user.
4. Create an object of the corresponding class based on the input.
5. Call the sound() method and display the appropriate sound, then stop the program.
